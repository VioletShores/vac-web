API Reference

Endpoints

POST /v1/sdk/signup Create developer account

No auth required. Returns your API key (shown once — save it immediately).

{ "status": "created", "api_key": "vac_live_a1b2c3d4...", "tier": "free", "limits": { "monthly_verifications": 100 } }
GET /v1/sdk/challenge Get verification challenge

Returns a random phrase and gesture. Prevents replay attacks. Expires in 120 seconds.

{ "challenge_phrase": "I am verifying my identity now", "challenge_gesture": 3, "expires_in_seconds": 120 }
POST /v1/sdk/verify Verify a human — core endpoint

Send a base64-encoded selfie video. Returns verification result with trust score. Costs 1 verification unit.

{ "verified": true, "trust_score": 0.847, "liveness": { "result": "passed", "score": 0.92 }, "deepfake": { "result": "passed", "score": 0.98 }, "voice": { "matched": true }, "gesture": { "detected": 3, "expected": 3, "matched": true }, "session_token": "vat_root_abc123...", "latency_ms": 2340 }
GET /v1/sdk/session/{token} Check session validity

Verify a session token is still valid in your backend. Use this to gate sensitive actions.

{ "valid": true, "trust_score": 0.847, "verified_at": "2026-03-09T10:23:45Z", "verification_method": "multi_modal_real" }
GET /v1/sdk/usage Check your usage stats

Returns current month's verification count, limit, and cost.

{ "billing_month": "2026-03", "verifications_used": 47, "verifications_limit": 100, "tier": "free" }
