What Is This?

This is a trust map of AI models. When you ask a question on SignalRank Live, Athena sends it to every available model simultaneously.

Some models agree. Some disagree. Some refuse to answer entirely. This graph visualises those relationships — so you can see at a glance which models trust each other and which diverge.

The more queries you run, the more accurate this map becomes. It's personalised to your usage.

How to Read the Graph
Circles = AI models. Larger circle = more trusted overall.
Number inside = Trust score out of 100, calculated from reliability, honesty, transparency, independence, and hallucination rate.
Solid green line = these two models gave similar answers. They agree.
Solid amber line = partial agreement. Models gave overlapping but not identical answers.
Dashed red line = these two models gave substantially different answers. They disagree.
Red dashed circle = this model has a high hallucination rate (often states incorrect things as fact).
Models close together = they tend to agree. Models far apart = they tend to diverge.

Restriction Map
When a model declines to answer or heavily hedges, Athena classifies the topic and tracks it here. This builds a map of which models restrict which subjects.
Topic categories:
political — elections, policy, political opinion
financial — investment advice, stock/crypto picks
medical — treatment recommendations, drug efficacy
conspiracy — contested events, conspiracy theories
personal_opinion — requests for the model's own opinion
unclassified — restriction detected but topic not yet categorised
Tag colours:
● Red tags — model has restricted this topic 3+ times (consistent pattern)
● Yellow tags — model has restricted this topic 1–2 times (emerging pattern)
