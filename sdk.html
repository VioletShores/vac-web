DEVELOPER SDK
Add cryptographic human provenance to any application
VAC Protocol sits underneath your existing identity stack. Verify humans, delegate authority to agents, enforce spending limits, and build trust graphs — all through a simple REST API.
THREE ENGINES, ONE API
Verification Engine
Multi-modal biometric verification. Pluggable — use our built-in engine or integrate iProov, Veriff, Sumsub, or your own provider. VAC handles the delegation chain regardless of who does the verification.
VAT Engine
Mint Verifiable Authority Tokens with scope, budget, and time constraints. Create delegation chains from human to agent to sub-agent. Monotonic narrowing enforced at every level.
Trust Graph
Build vouch networks where verification gets cheaper over time. Query trust scores, check vouch relationships, and let the graph drive your verification economics.
QUICK START
// 1. Verify a human
const auth = await fetch('https://api.vacprotocol.org/v1/vat/auth/verify', {
  method: 'POST',
  body: videoBlob // 6-modality biometric capture
});

// 2. Mint an authority token
const vat = await fetch('https://api.vacprotocol.org/v1/vat/delegate', {
  body: { role: 'researcher', budget_usd: 500, ttl_hours: 24 }
});

// 3. Agent acts within authority
const action = await agentExecute(vat.token, 'research task');

// 4. Provenance record — cryptographic proof of who authorised what
const proof = await fetch(`https://api.vacprotocol.org/v1/vat/verify/${vat.jti}`);
